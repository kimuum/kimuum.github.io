코드 규칙

들여쓰기

  • 1탭 들여쓰기 하며, 1탭의 크기는 공백 2칸으로 설정한다.
< div >
    < p >TEXT</ p >
< /div >

빈줄

  • 가독성을 고려하여 빈 줄을 추가한다. 단, 1줄을 초과하지 않는다.
< div >
    < p >TEXT</ p >
< /div >
 
< div >
    < p >TEXT</ p >
< /div >

class 입력

  • 케밥 케이스(word-word) 로 입력한다.
  • 클래스 네이밍은 넓은 범위부터 세부 범위 순서로 작성한다.
< div   class= "btn btn-large btn-primary btn-[유니크 클래스 이름]" > < /div >

기본 구조 wrapping

  • element의 최상단은 "-container" 로 선언한다.
< div   class= "content-container" > < /div >

CSS 작성 규칙

ID, Class 명을 작성 하고 띄어쓰기 후 스타일을 작성한다.

1
2
#id { width: "100%"; }
.class-name { width: "100%"; }

콜론(:)

콜론과 속성값 사이에 공백을 넣는다.

1
2
.class-name { margin-left :  0 ; padding-left :  10px; }

세미콜론(;)

선언 된 모든 속성에 세미콜론(;)을 사용한다.

1
2
.class-name { margin-left :   0 ; padding-left :   10px ; }

가상선택자 등록 방법

content에 작은 따옴표('')를 사용한다.

1
2
.class-name:after { content : ""; }

주석 / 빈줄

  • 주석이 필요한 경우 작성한다.
  • 종료 주석은 작성하지 않는다.
  • 코드 그룹 간 1줄의 빈 줄을 허용한다. 단, 1줄을 초과하지 않는다.
1
2
3
4
5
6
/* title */
.title-type   { margin :   30px   0   10px   0 }
 
/* description */
.desc   { padding :   2px   0   0   15px ; font-size :   16px }

공통 선택자

'*' 공통 선택자는 웹 페이지의 성능을 저하를 가져오기 때문에 가급적 사용하지 않는다.

1
2
3
/* 잘못된 예 */
* { margin :   0 ; padding :   0; }

나타나기

  • 기본 scroll-ani 클래스를 부여한다.
  • 슬라이드가 나타나는 방향에 대한 클래스를 부여한다.

  • ani-bottom : 아래에서 위로 나타나는 효과
  • ani-left : 왼쪽에서 오른쪽으로 나타나는 효과
  • ani-right : 오른쪽에서 왼쪽으로 나타나는 효과
