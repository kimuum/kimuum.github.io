데이터 주고받기 가입
데이터 주고받기란?

데이터 주고받기 결합을 활용하면, 통신요금을 더 알뜰하게 사용할 수 있어요.

데이터 주고받기는 같은 알뜰폰 사업자의 특정 요금제를 사용하는 경우 매월 2GB의 데이터를 주고받을 수 있는 서비스입니다. 데이터가 남는 가족 또는 친구가 있다면, 결합을 통해 매월 자동으로 2GB의 데이터를 주고받아 보세요.

가입 전 확인 사항
  • 데이터주고받기 결합은 같은 알뜰폰 사업자의 LTE 요금제인 경우 결합 한도 내에서 가능해요.
  • 데이터받기 요금제는 1회선만 결합 가능해요.
  • 결합 시, 주기 요금제의 기본제공량 2GB가 받기 요금제로 매월 이전되요. 단, 당월은 받기요금제에 프로모션 2GB 제공 후 익월(1일)부터 주기 요금제 기본제공량 2GB 차감 및 받기 요금제에 2GB 충전되요.
