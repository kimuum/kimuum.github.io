2. 데이터쉐어링이란?
  • 사용하고 있는 무제한 요금제에서 기본으로 제공하는 데이터를 추가요금 없이 태블릿, 또는 다른 스마트기기와 나눠 쓸 수 있는 서비스입니다. (1회선/무료)
  • 기본 제공량 초과 시 최대 200kbps 속도로 데이터 지속 사용 가능

3. 사용/신청 방법
  1. 데이터쉐어링이 사용 가능한 요금제인지 확인
  2. 유심칩을 준비
  3. 데이터쉐어링을 신청할 회선 정보를 넣고 데이터쉐어링 셀프개통
  4. 신청 시 사용하고 있는 모회선과 같은 명의로 쉐어링 요금제에 가입/개통되며, 유심을 사용하고자 하는 기기에 꽃아 사용

5. 유의사항
  • 데이터쉐어링은 1회선까지 등록 가능하며, 신청 시 무료로 제공됩니다.
  • 데이터쉐어링 회선의 데이터는 모회선의 기본 데이터 소진 후 200Kbps 속도로 지속 이용 가능합니다.
  • 모회선과 데이터쉐어링 회선의 명의자 및 다를 경우 데이터쉐어링 개통이 불가능합니다.
  • 모회선 또는 데이터쉐어링 회선의 해지/명의변경/일시정지/요금제 변경 시 연동이 해지되어 데이터쉐어링 이용이 불가합니다.
  • 데이터 쉐어링 회선은 가입 후 요금제 변경이나 번호이동으로 타사 이동이 불가하며 일반 해지만 가능합니다.
  • 데이터쉐어링 서비스는 추가 제공 데이터로는 사용이 불가합니다.
