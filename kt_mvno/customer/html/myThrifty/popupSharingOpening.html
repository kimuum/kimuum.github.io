데이터쉐어링 개통 처리

데이터 쉐어링 개통 처리중입니다.
데이터 쉐어링 개통 완료 처리가 안된 경우 셀프 개통 재요청 버튼을 눌러주세요.

데이터쉐어링 결합 실패 안내

데이터쉐어링 결합 신청이 실패하였습니다.
해당 회선 사업자 고객센터로 연락해주세요.

셀프개통 실패 안내

데이터쉐어링 대상(자회선) 셀프개통이 실패하였습니다.
다시 개통 요청을 시도해주세요.

데이터쉐어링 개통 신청완료

데이터 쉐어링 개통 신청이 완료되었습니다.
개통 처리 완료까지는 1~2분 정도 소요됩니다.

개통 진행 상태는 ‘개통신청조회' 화면에서 확인해주세요.
개통 완료 후 ‘데이터쉐어링 결합하기'를 진행해주셔야 사용이 가능합니다.
