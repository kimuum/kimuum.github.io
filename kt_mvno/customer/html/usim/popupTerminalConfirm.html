단말기 정보 확인 방법
iPhone, iPad 또는 iPod touch에서 모델명 또는 일련번호 찾기
  1. 설정 > 일반으로 이동하여 ‘정보’를 탭합니다.
  2. 일련 번호를 찾습니다.
  3. 이 정보를 Apple 등록 또는 지원 양식에 붙여 넣으려면 복사할 번호를 길게 터치합니다.
  • 자세한 내용은 애플 홈페이지를 확인해주세요.
제품 본체에서 확인 방법

휴대폰 구매 후 뒷면 하단에 일련번호가 각인되어 있습니다.
백글래스 파손 등으로 A/S를 받은 경우 교체 부품에는 일련번호가 각인되지 않습니다.
케이스를 교체하신 경우 설정 → 휴대전화 정보 → 규제 정보에서 확인 가능합니다.

휴대폰 메뉴에서 확인 방법

갤럭시S8 이후 출시 제품부터 설정 → 휴대전화 정보 → 규제 정보 → 일련번호 확인이 가능합니다.

  • 자세한 내용은 삼성 홈페이지를 확인해주세요.
