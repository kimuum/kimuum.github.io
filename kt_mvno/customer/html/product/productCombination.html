결합 이용안내
뭉칠수록 커지는 결합상품을 찾아보세요!

마이알뜰폰 고객이라면,
다양한 결합상품으로 더 합리적이게 소비할 수 있어요!

  • 무제한 요금제의 넘치는 데이터를 테블릿이나 다른 여분 단말에 무료로 나눠 사용할 수 있는 데이터 쉐어링!
  • 데이터가 남거나 혹은 부족하다면, 가족/지인과 결합하여 매월 데이터를 자동으로 주고 받는 데이터 주고받기!
데이터 쉐어링이란?

데이터 쉐어링은 대표회선의 데이터를 여러 기기에서 함께 사용할 수 있는 서비스로 대표 회선당 1개의 회선만 가입/결합하여 사용 가능합니다.

데이터 주고받기란?

데이터 주고받기 결합을 활용하면, 통신요금을 더 알뜰하게 사용할 수 있어요.


데이터 주고받기는 같은 알뜰폰 사업자의 특정 요금제를 사용하는 경우 매월 2GB의 데이터를 주고받을 수 있는 서비스입니다.
데이터가 남는 가족 또는 친구가 있다면, 결합을 통해 매월 자동으로 2GB의 데이터를 주고받아 보세요.
