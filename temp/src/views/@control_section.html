Control Section

영역 분할

  • 컨텐츠 내 상단 작은 단위의 요소들 버튼, 폼, 타이틀, 설명글, 리스트 영역 분할하는 Wrap 입니다. (class="control_section")
  • panel, panel_group 의 영역 밖에서 사용 중입니다.
  • 기본 우측 정렬
  • 내부 요소 2개일 경우 좌/우 정렬
  • control_section 의 내부에서는 btn_group 추가 클래스 사용하지 않아도 됩니다.
